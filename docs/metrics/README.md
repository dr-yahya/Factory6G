# Metrics Documentation

This directory contains all documentation related to simulation metrics.

## Files

1. **[01_overview.md](01_overview.md)** - Quick reference overview of all metrics
   - Currently implemented metrics
   - Critical missing metrics
   - Quick lookup table

2. **[02_implemented.md](02_implemented.md)** - Detailed documentation of implemented metrics
   - Complete list of implemented metrics
   - File naming conventions
   - Matrix/vector formats
   - Plot generation

3. **[03_physical_layer.md](03_physical_layer.md)** - Comprehensive physical layer metrics
   - Physical layer (PHY) metrics only
   - Detailed formulas and definitions
   - 6G requirements and targets
   - Implementation details

4. **[04_comprehensive_list.md](04_comprehensive_list.md)** - Complete metrics list
   - All metrics (implemented and planned)
   - Categorized by priority
   - 6G relevance and targets
   - Implementation recommendations

5. **[05_physical_layer_summary.md](05_physical_layer_summary.md)** - Physical layer metrics quick reference
   - Quick lookup for PHY metrics
   - Currently implemented
   - Critical missing metrics
   - 6G targets

## Reading Guide

- **New to metrics?** Start with `01_overview.md`
- **Want to know what's implemented?** Read `02_implemented.md`
- **Need PHY-specific details?** See `03_physical_layer.md` or `05_physical_layer_summary.md`
- **Planning new metrics?** Check `04_comprehensive_list.md`
